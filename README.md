## [描述](https://github.com/SoCXin/STM32L011/wiki)

[STM32L011](https://github.com/SoCXin/STM32L011)基于ARM Cortex®-M0内核，主要应用于低功耗场景，拥有常用的外设和各种规格的封装，适用于小尺寸设计。
